Example of parameter comm
this TBS template is W3C compliant

The comm parameter gives a supplementary comfort for designing a Template using a visual HTML editor (such as Dreamweaver or FrontPage). It enables you to enlarge the bounds of a locator to the bounds of the HTML commentary which surround it.

 

With the 'comm' parameter: It is now

Without the 'comm' parameter: It is now [onshow..now;frm='hh:nn:ss']

The merge result is identical with or without the 'comm' parameter. You can notice that the locator with the 'comm' parameter is visible only on the Template HTML source.
